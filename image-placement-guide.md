## DEPLOYMENT STEPS (Claude Code / Terminal)

1. Drop all .webp files into /images/ in the repo
2. Open each HTML file listed above
3. Find the section referenced and insert the <img> tag
4. Commit and push — Cloudflare Pages auto-deploys

## Notes
- Hero images: loading="eager" (above the fold)
- All other images: loading="lazy"
- All images include descriptive alt text for SEO
- Small images (471x630, 221x296) — use only as thumbnails/cards, not heroes
